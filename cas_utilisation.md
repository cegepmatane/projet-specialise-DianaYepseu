# Cas d’utilisation — CookieBook 🍪

## Acteur principal
Utilisateur (client)

---

## Cas 1 : Consulter le catalogue
### Description
L’utilisateur parcourt les cookies disponibles.

### Préconditions
- Accès au site

### Scénario principal
1. L’utilisateur arrive sur la page catalogue
2. Il consulte les cookies
3. Il sélectionne un cookie

---

## Cas 2 : Personnaliser un cookie
### Description
L’utilisateur personnalise son cookie.

### Préconditions
- Avoir sélectionné un cookie

### Scénario principal
1. Choisir la base du cookie
2. Ajouter des toppings
3. Visualiser le prix
4. Ajouter au panier

---

## Cas 3 : Ajouter au panier
### Description
L’utilisateur ajoute un produit au panier.

### Scénario principal
1. Cliquer sur "Ajouter au panier"
2. Le produit apparaît dans le panier

---

## Cas 4 : Passer une commande
### Description
L’utilisateur finalise son achat.

### Préconditions
- Avoir des produits dans le panier

### Scénario principal
1. Accéder au panier
2. Vérifier les produits
3. Entrer les informations (adresse, téléphone)
4. Confirmer la commande

---

## Cas 5 : Gérer son compte
### Description
L’utilisateur crée et gère son compte.

### Scénario principal
1. S’inscrire / se connecter
2. Consulter ses commandes
3. Modifier ses informations

# Risques du projet ⚠️

## Complexité technique
- Gestion du panier (state management)
- Synchronisation des données

## Sécurité
- Protection des données utilisateurs
- Gestion des paiements

## Temps de développement
- Beaucoup de fonctionnalités à implémenter
- Risque de dépassement de délais

## Bugs UX
- Mauvaise gestion du parcours utilisateur
- Confusion dans la personnalisation

## Dépendances
- Firebase ou API externe
- Risque de problèmes réseau

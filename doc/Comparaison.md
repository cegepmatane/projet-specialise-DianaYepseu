# Tableau comparatif

| Critères                      | Swiper.js                               | Google Maps API                        |
| ----------------------------- | --------------------------------------- | -------------------------------------- |
| Utilisabilité                 | Très facile à intégrer et à utiliser    | Plus complexe, nécessite configuration |
| UX & couleurs                 | Interface moderne, très personnalisable | Interface cartographique standard      |
| Librairies / modules          | Pagination, navigation, autoplay        | Cartes, marqueurs, géolocalisation     |
| Format (export et envoi)      | JavaScript, compatible React            | API Web JavaScript                     |
| Rapidité                      | Très fluide, animations légères         | Dépend du réseau et du chargement      |
| Pérennité                     | Projet activement maintenu              | Service durable de Google              |
| Coût                          | Gratuit                                 | Gratuit avec quotas                    |
| Licence                       | MIT                                     | Licence propriétaire                   |
| Interopérabilité (protocoles) | HTTP, API REST                          | HTTP, API REST                         |
| Sécurité                      | Pas de gestion de clés requise          | Clé API obligatoire                    |
| App internes                  | Très adapté aux apps web                | Peu adapté aux apps internes           |
| API                           | API simple orientée UI                  | API riche orientée services            |
| Automatisations               | Facilement automatisable                | Automatisations possibles via API      |

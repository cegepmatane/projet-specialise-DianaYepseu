# Fiche Projet — CookieBook 🍪

## Nom du projet
CookieBook — Site de vente de cookies personnalisés

## Description
CookieBook est une application web permettant aux utilisateurs de créer, personnaliser et acheter des cookies en ligne.
Le site offre une expérience fluide allant du choix des ingrédients jusqu’au paiement.

## Objectifs
- Offrir une expérience d’achat simple et intuitive
- Permettre la personnalisation complète des cookies
- Gérer les commandes et les utilisateurs
- Simuler un vrai site e-commerce

## Public cible
- Jeunes adultes
- Amateurs de pâtisserie
- Clients souhaitant offrir des cadeaux personnalisés

## Technologies utilisées
- Frontend : React
- Backend : Firebase / Node.js
- Base de données : Firestore / MongoDB
- Hébergement : Vercel / Netlify

## Fonctionnalités principales
- Catalogue de cookies
- Personnalisation des produits
- Panier d’achat
- Paiement
- Compte utilisateur

## Contraintes
- Interface responsive
- Sécurité des données utilisateurs
- Gestion des états (panier, session)

## Livrables
- Application web fonctionnelle
- Documentation technique
- Démonstration utilisateur
